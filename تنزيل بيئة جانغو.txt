
اولا يجب تحميل بايثون من موقعه الرسمي 
python.org/downloads

ويمكننا التأكد من التثبيت بالكتابة في التيرمنال
python --version

نسحب المشروع من git او ننزله كملف zip
git clone https://github.com/zena-am/Task-managment.git
ندخل لملف المشروع
cd Task-managment

من ثم من مجلد المشروع
pipenv install
pipenv shell/ هذه سنستخدمها كل ما ندخل المشروع لندخل البيئة الافتراضية للمشروع وبعدها نكتب التعليمات
